DAP-VSCODE-JS                                                    *dap-vscode-js*

Settings                                                            *Settings*
    Plugin configuration options

    Fields: ~
        {node_path}          (string)    Path of node executable. Defaults to
                                         $NODE_PATH, and then "node"
        {debugger_path}      (string)    Path to vscode-js-debug. Defaults to
                                         (runtimedir)/site/pack/packer/opt/vscode-js-debug
        {adapters}           (string[])  List of adapters to configure.
                                         Options are 'pwa-node', 'pwa-chrome',
                                         'pwa-msedge', 'node-terminal',
                                         'pwa-extensionHost'. Defaults to all.
                                         See
                                         https://github.com/microsoft/vscode-js-debug/blob/main/OPTIONS.md
                                         for configuration options.
        {defer_verification} (boolean)   Whether or not to force breakpoint
                                         verification to happen by the first
                                         continued event. This prevents
                                         breakpoints from flashing as rejected
                                         momentarily before being corrected,
                                         but may have some edge cases.
                                         Defaults to false.


dapjs.setup({settings})                                        *dapjs.setup()*
    Setup adapter and/or configs


    Parameters: ~
        {settings} (string)
